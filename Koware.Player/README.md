# Koware.Player

Cross-platform media player library/project used by the Koware ecosystem. Provides playback capabilities for streams opened via the CLI or other apps.

## Build

```bash
dotnet build Koware.Player/Koware.Player.csproj
```

## Notes
- Used by installer packaging to ship a bundled player.
- For Windows-specific packaging see `Koware.Player.Win`.
